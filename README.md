# 📚 Simple RAG QA System

A simple Retrieval-Augmented Generation (RAG) question answering system built with **LangChain, FAISS, and a local LLM**.

This project demonstrates how to build a lightweight **knowledge-based AI assistant** that retrieves relevant documents and generates answers using a large language model.

---

# 🚀 Features

- 📖 Custom knowledge base
- 🔎 Vector search with FAISS
- 🤖 Local LLM inference (Qwen / HuggingFace models)
- 🧠 Retrieval-Augmented Generation (RAG)
- 💬 Simple chat interface using Streamlit
